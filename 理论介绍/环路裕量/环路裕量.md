## 疑难解答  

### 1、    
![斜率和相位的计算](img/1.png)    
首先，解释下相位上的问题，H(S)中S = a+jw，这个是构成相位和幅值的基础    
H(S)=(S-Z1）（S-Z2）/(S-Z3）（S-Z4）     
在计算幅值的时候将（S-Z1）等视为向量，那么幅值就是|S-Z1|，相位就是e^Jθ,将所有的幅值和相位相乘和相除,那么就可以得到系统的幅值和相位.     
接着,我们对斜率进行解释:      
|H(jw)| ≈ sqrt(w^2 + z^2)→ 20*log10(|H|) = 10*log10(w^2 + z^2)   
当 w << z（远低于零点频率）：    
w^2 + z^2 ≈ z^2 → 幅值 ≈ 常数 → 斜率 = 0 dB/dec   

当 w >> z（远高于零点频率）：   
w^2 + z^2 ≈ w^2 → 幅值 ∝ w → 20*log10(w) → 斜率 = +20 dB/dec（注意X轴是10的指数，算斜率的时候要注意）   

在 w = z 处：实际值比渐近线高 +3 dB（转折点），斜率由 0 → +20 连续过渡。
所以：零点 z 是斜率「+20 dB/dec」的起始频率 —— 它“抬升”了高频增益，并让曲线变得更“陡”。
